## Software de Telemetria

O software de telemetria foi escrito para ser executado no Windows 10 ou Windows 11.

Para rodar, são necessárias as seguintes libraries:

* serial
* continuous_threading
* matplotlib
* drawnow
* pandas
* pillow (PIL)
